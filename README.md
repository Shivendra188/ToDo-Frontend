# Todo App (Vite + React)

## Setup
1. `npm install`
2. `npm run dev` (open http://localhost:5173)
3. `npm test`

## Features
- Create / Read / Update / Delete todos
- Subtasks, tags, priority, due dates
- Search, filter, sort, bulk actions
- Dark/light themes, compact/expanded view
- Persistence via localStorage + optional API adapter
- Jest + React Testing Library tests

## Notes
- CSS Modules used for component styles.
- To connect to a real server, replace `src/services/apiAdapter.js` with a real fetch/Axios adapter.
